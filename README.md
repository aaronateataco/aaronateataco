### 🔭 What I'm building

- **[Viridite](https://viridite.aaronworld.uk)** — my flagship project. A native compatibility layer that runs Android (NDK) games directly on the Nintendo Switch, no emulation. Custom ARM64 ELF loader, a dual-mapped SplitMap JIT to get around memory execution restrictions, and a growing per-game compatibility effort — built as an experimental human-AI collaboration with Claude. `C++`
- **[Equicord](https://equicord.org)** — the enhanced Vencord fork. I pitched the original concept and drew the first logos; it's grown into a feature-rich Discord client mod with 200+ plugins and a thriving community. `TypeScript`
- **[UserPFP](https://userpfp.vercel.app)** — animated Discord profile pictures for everyone, no Nitro required. Free, open source, adopted by thousands. `JavaScript`
- **[robloxbanwave](https://robloxbanwave.vercel.app)** — a real-time ban-wave radar for Roblox, powered by r/robloxhackers community reports, with 18 months of history and threat scoring. `JavaScript`
- **[Pets & Pals](https://modrinth.com/mod/pets-and-pals)** — a Minecraft mod giving more heart to your companions.
